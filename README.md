## 💻 Getting Started <a name="getting-started"></a>

> File uploader and tracker made with ExpressJs and Sequelize

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- [] NodeJs
- [] Postgresql

### Setup

Clone Back-end Repository:

```
git clone https://github.com/NatanimA/con-digital-api.git
```

Clone the front-end Repository

```
git clone https://github.com/NatanimA/con-digital-app.git
```


### Usage

To run the project for the Backend, execute the following command:

```
yarn install
```
OR

```
npm install
```

- First configure the port of your postgres configuration which is in config/config.json

- Then

```
npx sequelize-cli db:create
```

```
npx sequelize-cli db:migrate
```

```
yarn start
```

To Run the Project in the Frontend:

```
yarn install
```

```
yarn dev
```
